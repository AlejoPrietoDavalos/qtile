### Quickstart
Clonar el repo en `~/.config`.
```bash
cd ~/.config
git clone https://github.com/AlejoPrietoDavalos/qtile
cd qtile
```




### Archivo .log de qtile
Si se rompe algo, podés ver acá el error.
```bash
cat ~/.local/share/qtile/qtile.log
```
